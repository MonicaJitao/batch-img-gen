# 图像批量生成工作室 (Batch Image Generation Studio)

本项目是一个用于并发批量生成图像的本地工具集，包含一个现代化的 Web UI（Batch Studio）和一个用于快速测试的命令行脚本。通过本地 FastAPI 代理服务器，项目能够安全地转发请求至上游 API，解决跨域问题，并支持多张参考图的并发上传与任务轮询。

## ✨ 核心特性

- **现代化 Web UI**：提供直观的深色主题界面，支持并发生成、独立进度条展示和结果预览。
- **参考图支持**：支持上传最多 4 张参考图（PNG/JPEG/WebP），通过 `multipart/form-data` 安全转发至上游 API。
- **高并发与轮询**：基于异步请求（`httpx` + `asyncio`），支持一次性提交多个生成任务并并发轮询状态。
- **灵活配置**：支持自定义提示词、生成数量（1-20张）、多种图像比例（1:1, 16:9, 9:16 等）以及模型选择。
- **本地代理安全**：通过 FastAPI 构建本地代理，API Key 仅保存在本地 `.env` 文件中，不会暴露在浏览器前端。
- **一键下载**：支持单张图像下载及生成完成后的一键打包下载全部成功图像。

## 📁 项目结构

- `index.html` - 前端 Web 交互页面（Batch Studio）。
- `server.py` - 基于 FastAPI 的本地代理服务器，负责托管前端页面并转发 API 请求。
- `batch_generate.py` - 纯命令行的异步批量生成测试脚本。
- `requirements.txt` - Python 依赖清单。
- `VALIDATION.md` - 针对参考图上传和 API 链路的验收测试指南。
- `.env` - 环境变量配置文件（配置 API 密钥及基础 URL）。

## 🚀 快速开始

### 1. 环境准备

确保已安装 Python 3.8 或更高版本。

进入项目目录并安装依赖：

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

在项目根目录配置 `.env` 文件，填入你的 API 配置（如果尚未配置）：

```env
API_KEY=sk-your-api-key-here
API_BASE=https://api.tu-zi.com
```

### 3. 启动 Web 工作室 (推荐)

运行本地代理服务器：

```bash
uvicorn server:app --reload --host 127.0.0.1 --port 8765
```

启动后，在浏览器中访问 [http://127.0.0.1:8765](http://127.0.0.1:8765) 即可打开「Batch Studio」界面。

**使用说明**：
1. 在左侧面板输入提示词。
2. （可选）拖拽或选择上传参考图。
3. 选择生成数量和比例。
4. 点击「生成」，右侧面板将实时展示各槽位的任务进度。
5. 生成完毕后可单独下载或点击「全部下载」。

### 4. 使用命令行脚本测试

如果你只需要在终端中快速测试并发生成，可以直接运行：

```bash
python batch_generate.py
```

根据提示输入提示词和生成数量，脚本将自动并发提交任务并轮询打印进度，最终输出成功生成的图像 URL。

## 🧪 测试与验收

关于参考图功能、多文件上传（`multipart/form-data`）、比例字段映射等核心链路的详细测试方法，请参阅 [VALIDATION.md](./VALIDATION.md)。

## ⚠️ 注意事项

- **大批量 + 参考图**：当生成数量较大（如 20 张）且附带多张参考图时，由于每个并发任务都会重复上传整组参考图，可能会导致总上传体积较大、耗时增加。请留意上游 API 的并发限制和网络状况。
- **超时设置**：`server.py` 中针对带有参考图的请求（`/v1/videos`）已放宽了读写超时时间（300秒），以适应大文件上传和长耗时任务。
